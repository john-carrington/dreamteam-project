## Run Locally

**Prerequisites:**  Node.js


1. Install dependencies:
   `npm install`
2. (Optional) Set `VITE_API_BASE_URL` in [.env.local](.env.local) if API runs not on `http://localhost:8000`.
   For backends with prefixed API routes, use full prefix, for example `http://localhost:8000/api/v1`.
3. Run the app:
   `npm run dev`
